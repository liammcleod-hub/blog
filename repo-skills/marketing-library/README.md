# Marketing Skills Library

This folder is a repo-visible archive of the home-level marketing skills currently installed at:

- `~/.agents/skills`

It exists so the Bastelschachtel knowledge base is preserved in GitHub and can be reconstructed on another device without depending on one local machine.

## Purpose

This archive is for knowledge preservation, review, and future reconstruction.

It is not automatically the runtime location for skills. The normal active runtime path remains:

- `.agents/skills/...`

## Structure

- `skills/`: mirrored copies of the home-level installed skills
- `inventory.md`: index of the mirrored skill set

## Current Rule

- Runtime skill path for repo-local active skills: `.agents/skills`
- Repo-preserved library mirror: `repo-skills/marketing-library/skills`

If a new machine needs the same knowledge base, these folders can be copied back into the appropriate active skill path.
