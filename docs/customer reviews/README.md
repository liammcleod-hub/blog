# Customer Reviews

Read this file first when using `docs/customer reviews/`.

## Purpose

This folder is the Bastelschachtel VOC layer.

Use it to ground marketing, SEO, blog writing, category copy, and messaging in real customer language rather than assumptions.

## What Lives Here

- raw review exports and source files
- reusable VOC synthesis documents
- category-level VOC prioritization
- product-family language and proof points

## How To Use This Folder

Start here when you need:
- real phrasing customers use
- confidence triggers and objections
- product-outcome language
- category-specific proof points
- review-backed messaging angles

## Recommended Reading Order

1. This `README.md`
2. the most current category-level VOC map
3. the most current product-family VOC bank
4. the most current split/channel VOC bank
5. raw review files only when you need to verify or mine new language

## Working Rule

- Use synthesized VOC docs first.
- Use raw review exports second.
- Do not hard-code one dated file as the permanent source of truth; this folder will evolve.

## Typical Uses

- category-page copy
- product-page messaging
- blog posts
- SEO support content
- proof points for ads, email, and homepage work
