## Purpose

This folder is the Bastelschachtel SEO planning and execution layer.

Use it for article prioritization, category support strategy, landing-page direction, metadata framing, and internal-link logic.

## What Lives Here

- SEO audits
- execution checklists
- category copy direction
- master-article planning
- page-level messaging and search framing

## How To Use This Folder

Start here when you need:
- the current SEO priorities
- article or landing-page direction
- category support strategy
- internal-link structure
- search-intent framing for Bastelschachtel categories
