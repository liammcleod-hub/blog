# Bastelschachtel SEO Execution Checklist
#seo

*Based on: `docs/seo/live-seo-audit-2026-03-27.md`*
*Date: 2026-03-27*

## How To Use This

This is the practical execution layer for the live SEO audit.

Use it in this order:
1. Fix homepage positioning
2. Fix broken / missing category paths
3. Fix template-level H1 and metadata issues
4. Strengthen top commercial landing pages
5. Expand blog-topic coverage to support the category pages

## Phase 1: Critical Fixes

### 1. Homepage SEO repositioning
- [ ] Rewrite homepage title to target `Bastelbedarf Oesterreich` / `Kreativshop Oesterreich`
- [ ] Rewrite homepage H1 so it is not just the brand name
- [ ] Rewrite homepage meta description to include specialist categories and trust proof
- [ ] Review homepage hero copy so it balances brand tone with commercial search intent

**Target outcome:** Homepage becomes clearly rankable for Austrian craft-shop queries instead of reading like generic brand-first copy.

### 2. Peddigrohr / Korbflechten URL decision
- [ ] Confirm which URL should rank for Peddigrohr intent
- [ ] Make sure that URL returns `200`, not `404`
- [ ] Align the final page with these search targets:
- [ ] `Peddigrohr kaufen`
- [ ] `Korbflechten Material kaufen`
- [ ] `Peddigrohr Anfaenger Set`
- [ ] Add internal links to that page from homepage, `/pages/bastelbedarf`, and relevant blog posts

**Target outcome:** Basket-weaving intent points to one strong ranking page instead of a broken or unclear route.

## Phase 2: Template Repairs

### 3. Fix collection-template H1 behavior
- [ ] Check why some key collections do not expose a visible H1
- [ ] Fix collection template so every important collection outputs one real H1
- [ ] Verify this on:
- [ ] Pentart
- [ ] Bastelsets
- [ ] Reispapier
- [ ] Korbflechten / Peddigrohr

**Target outcome:** No major ranking page depends on a missing or decorative heading.

### 4. Fix blog index structure
- [ ] Add a real H1 to `/blogs/uebersicht`
- [ ] Rewrite the blog index meta description in clean German
- [ ] Add a short intro paragraph explaining what the blog covers

**Target outcome:** Blog archive becomes a clearer SEO entry point instead of just a listing page.

### 5. Standardize metadata for top collections
- [ ] Create a metadata pattern for collection pages:
- [ ] primary keyword first
- [ ] Austria or specialist framing where useful
- [ ] one clear benefit
- [ ] no clipping / broken endings
- [ ] Apply this pattern to:
- [ ] Reispapier
- [ ] Pentart
- [ ] Bastelsets
- [ ] Korbflechten / Peddigrohr
- [ ] Silikonformen fuer Gips und Beton

**Target outcome:** Top collection pages stop wasting title/meta real estate.

## Phase 3: Landing Page Expansion

### 6. Strengthen `/pages/bastelbedarf`
- [ ] Ensure the page has a visible H1
- [ ] Keep the long-form structure intact
- [ ] Add stronger internal links to Pentart, Reispapier, Korbflechten, and molds
- [ ] Treat this page as the reference template for future SEO landing pages

**Target outcome:** The strongest page becomes both stronger itself and the model for future pages.

### 7. Build or expand specialist landing pages
- [ ] Create or strengthen a `Pentart Oesterreich` page
- [ ] Create or strengthen a `Reispapier / Decoupage Reispapier kaufen` page
- [ ] Create or strengthen a `Korbflechten Material kaufen` page
- [ ] Create or strengthen a `Silikonformen fuer Gips und Beton` page
- [ ] Clarify whether `Bastelsets Oesterreich` should be a landing page, collection page, or both

**Target outcome:** Core commercial search themes each have a purpose-built destination.

## Phase 4: Content Support

### 8. Build category-linked tutorial clusters
- [ ] Publish or refresh Pentart tutorials
- [ ] Publish or refresh glass-etching tutorials
- [ ] Publish or refresh Decoupage / rice-paper tutorials
- [ ] Publish or refresh Korbflechten / Peddigrohr tutorials
- [ ] Publish or refresh gypsum / concrete casting tutorials

**Target outcome:** Informational content supports commercial category rankings instead of floating separately.

### 9. Improve internal linking between blog and money pages
- [ ] Add links from tutorial posts into the matching collection / landing pages
- [ ] Add links from category pages back to the most relevant tutorials
- [ ] Add links from homepage and `/pages/bastelbedarf` into the core specialist pages

**Target outcome:** Authority and relevance flow into the pages that should rank and convert.

## Phase 5: Schema & SERP Support

### 10. Expand structured data beyond the strongest page
- [ ] Keep Organization schema on the homepage
- [ ] Review whether homepage FAQ content should also expose FAQ schema
- [ ] Add or verify `BreadcrumbList` on top collections
- [ ] Add or verify page-level schema on important landing pages
- [ ] Recheck with a rich-results workflow after changes

**Target outcome:** Richer SERP eligibility on more than one page template.

## Success Markers

- [ ] Homepage no longer uses a brand-only H1
- [ ] No important specialist category returns 404
- [ ] All top collections have a visible H1
- [ ] Top collection metas are clean and keyword-directed
- [ ] At least one specialist landing page is stronger than the collection equivalent
- [ ] Blog content is clearly linked to revenue-driving categories
