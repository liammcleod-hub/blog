## 🎯 SEGMENTATION

**If you have reliable gender data:**
- Send Female versions to female subscribers
- Send Male versions to male subscribers

- Use Female versions for everyone (they're more neutral)
- OR pick the best subject line from each gender version

**Who to send to:**
- Subscribers engaged in last 90 days (opened or clicked)
- This protects your deliverability score
