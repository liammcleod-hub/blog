Use [[docs/reference/retool/README]] as the entrypoint for this folder.

This folder documents the current Bastelschachtel Retool workflow as a stable reference for future Codex skills, plugins, scripts, and QA automation.

These docs are intended to preserve the real operating process as it exists now, even if parts of the workflow later become more automated.

Retool is the active operator interface.

Codex should treat these Retool outputs as first-class pipeline artifacts, not as ad hoc pasted context.
