# Blog SEO Pipeline QA Report Template
#content-pipeline #blog

Use this when evaluating a Retool-generated article.

## Findings

Present findings first, ordered by severity.

Each finding should include:

- issue
- evidence
- risk
- fix

## Coverage Checks

Evaluate:

- dossier grounding
- brief alignment
- product-link integrity
- SEO coverage
- format match
- source integrity
- information gain

## Priority Fixes

List the smallest set of changes needed before publishing.

## Residual Risks

List anything that cannot be fully verified from the available artifacts.
